## How to Download

You can download these images using:

1. **Manual Download**: Right-click on each URL above and save the image
2. **Using curl** (from terminal in this directory):
   ```bash
   curl -o ird-logo.png https://ird.gov.np/public/images/1168015708.png
   curl -o modal-image.jpg https://ird.gov.np/public/images/869883628.jpg
   curl -o ird-app-banner.jpg https://ird.gov.np/public/images/1397622153.jpg
   curl -o basudev-poudel.jpg https://ird.gov.np/public/images/1311898185.JPG
   ```

3. **Using wget**:
   ```bash
   wget -O ird-logo.png https://ird.gov.np/public/images/1168015708.png
   wget -O modal-image.jpg https://ird.gov.np/public/images/869883628.jpg
   wget -O ird-app-banner.jpg https://ird.gov.np/public/images/1397622153.jpg
   wget -O basudev-poudel.jpg https://ird.gov.np/public/images/1311898185.JPG
   ```
