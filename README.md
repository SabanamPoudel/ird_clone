# IRD Taxpayer Portal Demo

This is a **legitimate web application** for the Inland Revenue Department (IRD) Nepal Taxpayer Portal demonstration.

## About This Project

This repository contains HTML, CSS, and JavaScript files for a taxpayer portal system. It includes:

- Tax return entry forms (D-01, D-02, D-03, D-04)
- VAT return management
- E-TDS (Electronic Tax Deducted at Source) system
- Payment voucher system
- Taxpayer login and registration

## Security Notice

**This is NOT malware or a virus.** Some antivirus software may flag this repository due to:
- Form input handling code patterns
- Login form structures
- JavaScript form validation
- Multiple HTML files with similar structures

These are standard web development practices for creating tax filing systems.

## File Structure


## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/suzwon/ird-demo.git
   ```

2. Open `html/index.html` or `html/taxpayer_portal.html` in a web browser

## False Positive Virus Detection

If your antivirus flags this repository, you can:
1. Verify the code is safe by reviewing the source files
2. Add the repository to your antivirus whitelist
3. Download directly from GitHub instead of as a ZIP file
4. Use `git clone` instead of downloading

All code is open source and can be inspected for safety.
