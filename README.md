# IRD Nepal Website Replica

This is an exact replica of the Internal Revenue Department (IRD) Nepal website homepage.

## Files Included:
- `index.html` - Main HTML structure
- `style.css` - Complete styling with exact colors and layout
- `logo.png` - Nepal Government logo (SVG)
- `phone.png` - Phone mockup for app showcase (SVG)
- `appstore.png` - App Store badge (SVG)
- `playstore.png` - Google Play badge (SVG)

## Features:
✅ Exact color scheme matching the original
✅ Government of Nepal logo
✅ Full navigation menu in Nepali
✅ Interactive buttons (OFFICER PORTAL, IRD MAIL, ARCHIVES, etc.)
✅ Search functionality layout
✅ Large promotional banner for IRD Nepal App
✅ Feature list with yellow badges
✅ Phone mockup showcase
✅ App store download buttons
✅ Bottom service menu with 6 items
✅ Right sidebar with 8 service boxes
✅ Responsive design
✅ Hover effects on all interactive elements

## Color Scheme:
- Primary Blue: #1a9fd8 (navigation and service boxes)
- Red: #c41e3a (government logo text)
- Light Blue: #7dd3e8 to #5bc3dd (gradient background)
- Yellow-Green: #d4f169 (feature badges)

## To View:
Simply open `index.html` in any modern web browser.

## Notes:
- All text is in Nepali (Devanagari script)
- Uses Mukta font family for proper Nepali text rendering
- Fully responsive layout
- All images are SVG-based for scalability
